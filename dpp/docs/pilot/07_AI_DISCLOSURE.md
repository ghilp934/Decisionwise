## 서비스 AI 사용 고지

### 기본 원칙

Decisionproof API Platform은 **생성형 AI 기술**을 사용하여 Decision Pack을 실행합니다.

**사용 범위:**
- 입력 데이터 분석
- 의사결정 결과 생성
- 추천 및 인사이트 제공

**모델 정보:**
- 주요 모델: [모델명 - 별도 공개 시]
- 학습 데이터: 공개 데이터셋 및 자체 데이터
- 업데이트: 수시 (성능 개선)

## 고객의 AI 고지 의무

### 산출물 사용 시 고지 권장

파일럿 고객이 Decisionproof API의 **산출물을 재배포**하는 경우, 아래 방식으로 생성형 AI 사용을 고지하는 것을 권장합니다.

#### 방법 1: 가시적 표시 (Visible)

**예시:**
```
본 내용은 생성형 AI 기술(Decisionproof API)을 사용하여 생성되었습니다.
```

**표시 위치:**
- 웹페이지 하단 (푸터)
- 문서 말미
- 앱 설정 > 정보

#### 방법 2: 메타데이터 표시 (Invisible)

**예시 (HTML):**
```html
<meta name="generator" content="Decisionproof API v0.4.2.2">
<meta name="ai-generated" content="true">
```

**예시 (JSON):**
```json
{
  "content": "...",
  "_metadata": {
    "generated_by": "Decisionproof API",
    "ai_model": "decision-v0.4",
    "generated_at": "2026-02-15T12:34:56Z"
  }
}
```

#### 방법 3: 이용 약관에 명시

**예시:**
```
본 서비스는 의사결정 보조를 위해 Decisionproof API(생성형 AI)를 사용합니다.
생성된 결과는 참고용이며, 최종 결정은 사용자의 책임입니다.
```

## 고지 체크리스트 (고객용)

고객이 산출물을 재배포하기 전에 아래 항목을 확인하세요:

- [ ] **AI 사용 여부 명시**: "생성형 AI 사용" 문구 포함
- [ ] **출처 표시**: "Decisionproof API" 또는 "당사 AI 시스템" 명시
- [ ] **책임 한계**: "참고용이며 최종 결정은 사용자 책임" 고지
- [ ] **업데이트 가능성**: "AI 모델 업데이트로 결과 변동 가능" 안내 (선택)

## 면책 및 한계

### AI 산출물 한계

**주의사항:**
- [ ] AI 생성 결과는 참고용 (절대적 정확성 보장 안 함)
- [ ] 최종 의사결정은 인간이 수행 권장
- [ ] 중요 결정(법률/의료/금융)은 전문가 검증 필수

### 면책 조항 예시

고객이 자사 서비스에 포함할 면책 조항 예시:

```
본 서비스의 AI 생성 결과는 참고용이며, 정확성이나 적합성을 보장하지 않습니다.
중요한 의사결정은 반드시 전문가와 상담하시기 바랍니다.
AI 결과 사용으로 인한 손실/손해에 대해 당사는 책임지지 않습니다.
```
