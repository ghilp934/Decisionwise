Private Beta — 4 steps

From sign-up to first API call

No SDK required. No OAuth flow. Just a REST endpoint, an API key, and a structured decision — ready in minutes.

1

Create your account

Sign up at decisionproof.io.kr with your email address. Account creation takes under a minute — no credit card required at this step.

2

Subscribe for 30-day access

Pay $29 via PayPal to activate your account for 30 days. Each paid beta cycle includes 1,000 decision credits with a hard cap of 2,000. Access does not auto-renew; you choose when to renew.

See full pricing details →

3

Create an API key

Once your payment is confirmed, your onboarding dashboard activates. Create up to 3 API keys for your workspace during beta — each key can be revoked independently at any time. No OAuth scopes to configure.

4

Submit your first decision

POST a structured decision payload to the API. Get back a deterministic, auditable result. Every run is logged — export the full decision trace at any time. 

curl -X POST https://api.decisionproof.io.kr/v1/decisions \
  -H "Authorization: Bearer sk_live_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "decision_type": "approval",
    "inputs": { "score": 87, "threshold": 80 }
  }'
